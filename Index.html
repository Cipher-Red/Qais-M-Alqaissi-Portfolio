Caeruleum

Caeruleum is a lightweight, Python-based command-line tool for forensic analysis on Windows. It offers essential utilities like file recovery, malware scanning, system log collection, disk management, and network scanning. Designed for both novice and experienced users, it’s resource-efficient and compatible with low-resource systems.

Key features include deleted file recovery, forensic image generation, Windows Defender malware scans, and detailed system health reports. Open-source and licensed under Apache 2.0, Caeruleum is a versatile tool for forensic investigations and system analysis.

Vigil

Vigil is a lightweight, open-source network traffic monitoring tool built in Python. It captures and analyzes live network traffic in real-time, offering features like protocol identification, IP/port filtering, DoS/DDoS detection, and payload inspection. Captured packets are saved in PCAP format for further analysis.

Designed for network administrators and security researchers, Vigil supports cross-platform use (Linux and Windows) and is licensed under Apache 2.0. Its intuitive interface and powerful capabilities make it ideal for monitoring, troubleshooting, and securing network environments.
